# Roadmap

The roadmap lives in GitHub, where it stays in sync with the work instead of drifting in prose:

- [Milestones](https://github.com/lathrys-at/shrike/milestones): one per minor version, each tracked by an epic issue.
- [Issues](https://github.com/lathrys-at/shrike/issues): open and planned work.

See [`CONTRIBUTING.md`](../CONTRIBUTING.md) for how the roadmap is organised.
